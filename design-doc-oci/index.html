Oracle Cloud 設計書作成ツール(仮)

やり方

１、適当な読み取り専用の OCI ユーザを作る
２、適当なグループに所属させる
３、読み取り専用のポリシーを適用する
　　(例：ALLOW GROUP ReadOnly_Group to read all-resources in compartment ctc-yamada)
４、下のボタンを押して公開鍵を入手して全てコピーする
　　※鍵の有効時間は 1 時間なので注意！※

５、OCI ユーザの「API キー」に生成した公開鍵(先ほどコピーしたもの)を登録する
　　(フィンガープリントはメモしておく)
６、3 分くらい待つ
　　(OCI が認識するのに時間がかかるようでこれをしないと情報が正しくても失敗する)
７、以下のそれぞれの値を調べて入力してボタンを押す

User OCID :
Tenancy OCID :
Fingerprint :
Compartment OCID :
Namespace :
Region :
